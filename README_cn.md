Pluribus 片头动画复刻 (Pluribus Title Sequence Recreation)

本项目使用 Web 前端技术（HTML5 Canvas 和原生 JavaScript），复刻了美剧 Pluribus 的片头动画效果。

**任务目标**：使用 HTML5 Canvas 和原生 JavaScript 单文件实现美剧《Pluribus》片头粒子动画特效。

**核心功能与视觉要求：**

1.  **排版与字体**：
    * 引入 Google Fonts 的 `Montserrat` (Black 900) 字体。
    * 在屏幕中心渲染文字 "PLUR1BUS"。
    * **磁场纹理**：文字内部的粒子不使用普通网格排列，而是基于以首字母 'P' 为中心的**椭圆磁场公式**进行排列（带有微小的正弦波有机扰动），形成类似指纹的向心纹理。
    * **负空间**：计算每个字母的几何位置，在字母周围建立椭圆形的**排斥区**，强制背景粒子无法进入这些区域，确保文字边缘清晰锐利。

2.  **背景粒子系统**：
    * 生成覆盖全屏的高密度白色粒子网格。
    * **物理属性**：背景粒子应表现为“刚性介质”，具有极高的回弹系数（Friction > 0.85）。受到扰动后应瞬间弹回原位，而不是漂移。

3.  **声波物理模拟**：
    * 以 'P' 字母为震源，高频发射圆形的物理脉冲波。
    * **聚集效应**：声波经过时，**不要**推开粒子，而是将周围的背景粒子强力**吸向波峰**，形成高密度的压缩带。
    * **视觉渲染**：不要绘制线条。声波的可视化应完全依靠粒子的**密度叠加**来呈现。为了防止物理移动产生黑色空隙，需在波峰位置动态注入临时的“幽灵粒子”，其大小、亮度和密度需经过数学计算，与背景粒子完美融合。
    * **文字稳定性**：文字粒子几乎不受声波物理推力的影响，保持稳定，仅保留持续的微幅电流噪点抖动。

4.  **音频与交互**：
    * 加载并循环播放 `pluribus_music.mp3`。
    * **自动播放处理**：添加一个全屏交互遮罩层（Overlay），提示用户“点击屏幕初始化系统”。用户点击后，移除遮罩并开始播放音频和动画，以解决现代浏览器的自动播放限制。

5.  **代码要求**：
    * 代码需包含在一个 HTML 文件中。
    * 确保性能优化，能流畅处理数千个粒子的实时物理计算。
